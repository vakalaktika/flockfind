# FlockFind

A responsive product-comparison app for flamingo and style-matched wild-bird peel-and-stick wallpaper sold through U.S. storefronts. It normalizes known prices to square feet, adjusts ratings for review sample size, scores visible quality signals, and estimates full project cost.

## Run locally

Open `index.html` directly, or serve the folder with any small local web server.

No build step or third-party packages are required.

## Product data

The catalog contains 81 distinct patterns:

- 36 flamingo products
- 45 wild-bird products with tropical, jungle, or mid-century-modern styling
- 26 manufacturers and 10 U.S. retailers or manufacturer storefronts

It is a hand-curated snapshot of linked listings checked June 27, 2026. Known roll prices and coverage are normalized to price per square foot. Listings with incomplete price or product-level review evidence are explicitly labeled instead of estimated. Prices, ratings, inventory, shipping, and returns can change, so shoppers should verify details on the linked source.

## Accessibility

The interface targets WCAG 2.2 Level AA, the current published W3C recommendation. It includes semantic native controls, complete keyboard operation, visible focus, live result announcements, accessible modal dialogs, reduced-motion support, forced-color support, responsive reflow, text alternatives, and contrast-safe interaction states.

Retailer pages open in a new tab and have independent accessibility policies.
